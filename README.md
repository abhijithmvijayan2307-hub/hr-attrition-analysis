# HR Employee Attrition Analysis
End-to-end Data Analytics Portfolio Project

## Tools Used
- Python, pandas, numpy, matplotlib, seaborn
- Scikit-learn, XGBoost, SHAP
- Power BI
- Jupyter Notebook

## Key Results
- XGBoost model achieved 74% Recall and 77.5% ROC-AUC
- 435 high risk employees identified
- 3-page interactive Power BI dashboard built
